# robot
这是一个人机对话的项目

这个项目是基于SpringBoot和Thymeleaf开发的项目。

# 使用说明
把SendRequestService类中的userInfo.setApiKey("169e1daebd7a4138a7ec36e09af9aa43")的apiKay值改为自己apiKey。
这个apiKey可以从图灵中获取。先要去http://www.turingapi.com/ 申请图灵机器人。
